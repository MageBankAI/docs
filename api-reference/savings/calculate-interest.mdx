---
title: 'Calculate Interest'
api: 'POST /investment/calculator'
description: 'Calculate potential interest for a given investment amount and period'
---

<Note>
**Investment Planning**: Project potential earnings on investments with detailed calculation breakdowns for financial planning.
</Note>

## Request Headers

<ParamField header="x-api-key" type="string" required>
  Your API key for authentication
</ParamField>

## Request Body

<ParamField body="amount" type="number" required>
  Principal amount to invest
</ParamField>

<ParamField body="days" type="number" required>
  Investment duration in days
</ParamField>

<ParamField body="customRate" type="number">
  Optional custom interest rate (annual percentage)
</ParamField>

## Response

<ResponseField name="principal" type="number">
  Principal amount for the calculation
</ResponseField>

<ResponseField name="interestRate" type="number">
  Annual interest rate applied to the calculation
</ResponseField>

<ResponseField name="interestEarned" type="string">
  Interest amount earned over the period
</ResponseField>

<ResponseField name="totalAmount" type="string">
  Total amount including principal and interest
</ResponseField>

<ResponseField name="calculation" type="object">
  Formula and step-by-step calculation process
</ResponseField>

<ResponseExample>
```json 200 
{
  "principal": 1000,
  "days": 365,
  "interestRate": 4.5,
  "interestEarned": "45.00",
  "totalAmount": "1045.00",
  "annualYield": "4.50%",
  "calculation": {
    "formula": "principal × rate × (days ÷ 365)",
    "steps": [
      "1000 × 0.045 × 365/365",
      "1000 × 0.045 × 1.000000",
      "1000 × 0.045 × 0.045000",
      "45.000000"
    ]
  }
}
```

```json 400 
{
  "success": false,
  "error": "Amount and days are required"
}
```

```json 400 
{
  "success": false,
  "error": "Amount must be a positive number"
}
```
</ResponseExample>

## Status Codes

| Status Code | Description |
|-------------|-------------|
| 200 | Interest calculation completed successfully |
| 400 | Bad Request - Invalid or missing parameters |
| 500 | Internal Server Error - Error fetching interest rate or calculating interest |

<Info>
**Current Rate**: If no custom rate is provided, the calculation will use the platform's current interest rate fetched from the database.
</Info>

<Info>
**Formula Used**: Interest is calculated using the simple interest formula: `principal × rate × (days ÷ 365)`.
</Info>

<Check>
**Financial Planning**: You can compare different investment scenarios by adjusting the amount, duration, and custom rate parameters.
</Check>
