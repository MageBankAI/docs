# Savings Vault

The MageBank Savings Vault allows you to earn yield on your USDC holdings while maintaining liquidity for your agents.

## About Savings Vault

The Savings Vault provides:

-   Competitive yield on your USDC deposits
-   Instant liquidity for your agents
-   Automated optimization of idle funds
-   Detailed analytics on earnings

## Getting Started with Savings Vault

<Steps>
    <Step title="Access Savings Vault">
        Navigate to the [Savings
        Vault](https://www.magebank.ai/dashboard/savings-vault) section from the
        sidebar menu.
        <Frame>
            <img
                src="/images/savings-vault-menu.png"
                alt="Savings Vault Menu"
            />
        </Frame>
    </Step>

    <Step title="Activate Your Vault">
        If this is your first time, click "Activate Vault" to set up your
        savings account.
        <Frame>
            <img src="/images/activate-vault.png" alt="Activate Vault Button" />
        </Frame>
    </Step>

    <Step title="Deposit Funds">
        Click "Deposit" to add funds to your Savings Vault.
        <Note>
            You can deposit directly from your connected bank account or
            transfer from your agent balances.
        </Note>
        <Frame>
            <img src="/images/vault-deposit.png" alt="Vault Deposit Form" />
        </Frame>
    </Step>

    <Step title="Configure Settings">
        Set your preferences: - **Auto-Optimization**: Automatically move idle
        funds to earn yield - **Liquidity Threshold**: Minimum balance to keep
        available for transactions - **Yield Strategy**: Conservative, Balanced,
        or Growth
    </Step>
</Steps>

## Vault Management

The Savings Vault dashboard provides comprehensive information:

-   **Current Balance**: Total funds in your vault
-   **Earned Interest**: Total yield earned to date
-   **Current APY**: Annual percentage yield being earned
-   **Transaction History**: Record of deposits and withdrawals

## Withdrawing Funds

<Steps>
    <Step title="Initiate Withdrawal">
        Click the "Withdraw" button on your Savings Vault dashboard.
        <Frame>
            <img src="/images/vault-withdraw.png" alt="Withdraw Button" />
        </Frame>
    </Step>

    <Step title="Specify Details">
        Enter withdrawal amount and select destination: - Transfer to an agent
        balance - Withdraw to external wallet - Transfer to bank account
    </Step>

    <Step title="Confirm Withdrawal">
        Review the details and confirm your withdrawal.
        <Note>
            Standard withdrawals process within 1 business day. Express
            withdrawals are available for a small fee.
        </Note>
    </Step>
</Steps>

## API Access to Savings Vault

Manage your Savings Vault programmatically:

```javascript
// Example: Check Vault Balance
const getVaultBalance = async () => {
    const response = await fetch(
        "https://api.magebank.ai/v1/savings-vault/balance",
        {
            method: "GET",
            headers: {
                Authorization: `Bearer ${process.env.PAYMAN_API_SECRET}`,
            },
        }
    );

    return await response.json();
};

// Example: Deposit to Vault from Agent
const depositToVault = async (agentId, amount) => {
    const response = await fetch(
        "https://api.magebank.ai/v1/savings-vault/deposit",
        {
            method: "POST",
            headers: {
                "Content-Type": "application/json",
                Authorization: `Bearer ${process.env.PAYMAN_API_SECRET}`,
            },
            body: JSON.stringify({
                agent_id: agentId,
                amount: amount,
                currency: "USDC",
            }),
        }
    );

    return await response.json();
};
```
